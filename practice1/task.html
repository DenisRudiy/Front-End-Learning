Завдання на практичну роботу 1

  1. Встановити VS code, git
  2. Створити репозиторій на GitHub
  3. Створити та налаштувати папку для дисципліни, вкладені папки для лабораторних та практичних
  4. Створити локальний репозиторій
  5. Створити гілки для роботи (work) та звітності (ready)
  6. В робочій гілці у папці practice1 створити .html файл, виконати commit
  7. Пов'язати VS code with GitHub
  8. Пов'язати локальний репозиторій з глобальним
  9. Виконати push
  10. Перейти у гілку ready виконати merge
