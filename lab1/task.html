Лабораторна робота 1

Завдання на лабораторну роботу 1

  1. В робочій гілці власного репозиторію створити папку lab1 для виконання роботи
  2. Обрати шаблон сторінки резюме у графічному вигляді
  3. Здійснити декомпозицію шаблону за допомогою html із дотриманням наступних вимог
    • Відповідність обраному шаблону
    • Використання тегів div, img, ul, form, input тощо
    • Наявність форми зворотнього зв'язку
    • Наявність меню, підвалу сторінки (footer)
    • Можливість переходу на сторінку з фото обраного шаблону
  4. Під час виконання лабораторної роботи забороняється використовувати стилі
  5. Виконати commit, merge (гілка ready), push
  6. Підготуватися до захисту роботи
